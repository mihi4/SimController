# SimController

Based vaguely on BMSAIT I decided to write my own reader and arduino communications application.  
BMSAIT's data updates were a bit too slow for my servo solution and the communications protocol too big to send data fast between PC and Arduino.

Big THANK YOU to Hummer for BMSAIT, parts of the Arduino controller code are based on his code.

This project is not supposed to be an "fit-all" type of datareader, it's completely designed to fit my used hardware.  
But it can be a nice starting point for anybody...
